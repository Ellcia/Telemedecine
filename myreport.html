NeuroKit Processing Report

Preprocessing

The raw signal, sampled at 50 Hz, linear detrending followed by a fifth order 2Hz low-pass IIR Butterworth filter; following Khoadadad et al., 2018.
The peak detection was carried out using the method described in Khoadadad et al. (2018).
