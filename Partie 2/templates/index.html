Interface réalisée par Alice Guilloteau, dans le cadre de l’unité 23_E5_SEV_5106E - Environnement de télémédecine pour l'acquisition et le traitement de données physiologiques et cliniques à ESIEE Paris.

Cette interface utilise les framework Flask ainsi que Bootstrap. Les traitements et analyses des signaux respiratoires ont été effectués avec la librairie Neurokit2.

Le Student Id correspond à un identifiant de type « E1 », la Session Id un identifiant de type « S1 » et la Sequence Id à un entier. Pour plus d’informations, se référer à la documentation du serveur stockant les données : https://www.gaalactic.fr/~sev_5106e/help#getRequestUriSpec
